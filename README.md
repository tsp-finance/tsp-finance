## 🚀 What is TSP?

**Trading System Platform** is a web-based application that empowers traders and developers to create algorithmic trading strategies using a proprietary scripting language, validate them against historical market data, and deploy automated bots that execute 24/7.

TSP is currently in active development. You can **[join the waitlist](https://www.tsp.finance/signup)** to get early access.

## ✨ Features

| | Feature | Description |
|---|---|---|
| 📝 | **Strategy Builder** | Write and validate trading strategies with a built-in code editor and real-time syntax checking |
| 📊 | **Backtesting** | Simulate strategies against historical OHLCV data with detailed performance reports |
| 🤖 | **Automated Trading** | Deploy bots connected to exchanges, with real-time monitoring and trade tracking |
| 🕯️ | **Market Analysis** | Interactive candlestick charts with multiple timeframes, technical indicators, and bidirectional data loading |
| 📈 | **Performance Reports** | Profit/loss breakdowns, cost analysis, equity curves, and key performance metrics |
| 🔒 | **Enterprise Security** | AWS Cognito authentication, encrypted API key management, and GDPR compliance |
